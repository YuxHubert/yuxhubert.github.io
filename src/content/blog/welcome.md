这里是一个新的个人博客。第一版的目标很简单：把内容写在 Markdown 里，把网站交给 GitHub Pages 自动部署。

Markdown 适合长期写作，因为它足够轻，文件也容易迁移。以后可以把这里改成真正的自我介绍、项目记录、读书笔记，或者任何值得沉淀的想法。

这篇文章只是一个起点。真正重要的是形成一个稳定的发布流程：写下内容，提交代码，推送仓库，然后让自动化流程完成上线。
